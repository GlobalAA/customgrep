## Mini grep program
This program created during the study golang!

### Usage
#### First Step
```bash
    make
```
#### Second Step
```bash
    ./grep
```

#### Params

| Params        | Description      | Default Value  |
|:------------- |:-----------------|:---------------|
| grep          | What to find     | NULL           |
| path          | File path to open| NULL           |


### Run example
```bash
    ./grep -grep hello -path main.txt
```
